Burren Outdoor Education Centre

Das Burren Outdoor Education Centre wurde 1982 von County Clare V.E.C. gegründet und war eines der ersten dezidierten "outdoor education center" in Irland. Seither erorschten zehntausende Besucher die einzigartige Landschaft von Burren.

Die Kombination von Einrichtungen zum akademischem Lernen einerseits und Sport/outdoor Aktivitäten andererseits macht das Burren Outdoor Education Centre zum idealen Ort fü um Programmieren zu lernen und intensiv die Natur zu erleben.

Scratch

Scratch ist eine visuelle Programmiersprache entwickelt vom Massachusetts Institute of Technology Media Lab speziell für Kinder. Scratch ist freie (free/libre open source) Software. Im Gegensatz zu anderen Programmiersprachen ist bei Scratch kein Tippen notwendig. Staddtedessen werden vorgefertigte Code-Blöcke zusammengeschoben, ähnlich wie bei einem Puzzle-Spiel. Scratch funktioniert online und offline. Scratch ermöglicht das Erstellen von Spielen, Präsentationen, Simulationen, Animationen und Kunstprojekten. Scratch ist das ideale Werkzeug um Programmieren ohne Vorkenntnisse zu lernen und erfreut sich einer wachsenden weltweiten Community von Scratchern aller Altersgruppen

Python

Python ist eine "echte" allzweck Programmiersprache. Python ist freie (free/libre open source) Software. Python ist downloadbar für alle bekannten Betriebssysteme (Linux, Windows, Mac). Bei Linux (Ubuntu) und Mac Rechnern ist Python vorinstalliert, ebenso beim RaspberryPi. Im Gegensatz zu anderen Programmiersprachen ist Python nicht hochtspezialisiert auf ein bestimmtes Einsatzgebiet. Python ist bekannt dafür leicht erlernbar zu sein und gut lesbaren (schönen) Code zu erzeugen. Python wird von immer mehr Universitäten weltweit eingesetzt als Einstiegs-Programmiersprache für Studenten
Python selbst erlaubt text-basierte Spieleprogrammierung sowie einfache Grafik-Programme dank Python's mitgeliefertem Turtlegrafk-Modul. Für Python existieren mehrere spezialisierte game-engines:

Pygame ist eine externe Python-Erweiterung speziell zum Erstellen von 2D Spielen. Pygame ist freie (free/libre open source) Software. Mit Pygame kann man alle Arten von 2D games mit Sound, Grafik, Joystickunterstützung etc. programmieren. Horst JENS benutzt python/pygame seit 2006 zur Spiele-Programmierung mit seinen Kursteilnehmern.

Blender

Blender ist ein 3D Modeling/Animations-Programm. Blender ist freie (free/libre open source) Software. Blender erlaubt das Erstellen von 3D-Models, Animationen und Spielen. Blender kann mit Hilfe von Python programmiert werden. Blender hat zusätzlich eine eingebaute 3D game engine die hautsächlich mit visuellen Programmiertechniken funktioniert.

Web Design

Keine Programmier-Vorkenntnisse erforderlich. Die Kursteilnehmer lernen eine eigene Website zu bauen. Dabei lernen sie u.a.:

  • Arbeiten mit Texteditoren (copy/paste/search/replace).
  • Arbeiten mit Web development tools und Frameworks.
  • Eine Website von Grund auf zu designen und zu bauen.
  • Online resources für Web-development finden
  • Web development Workflow.
  • Arbeiten mit professionell erstellten style templates aus dem Internet

Pocket Code

Pocket Code und Pocket Paint sind Android Apps zum Programmieren einfacher Apps und Spiele auf Android Smartphones/Tablets. Pocket code und Pocket Paint sind freie (free/libre open source) Software. Pocket Code verwendet visuelle Programmiersprachen-Elemente, ähnlich wie Scratch, aber speziell angepasst für Smartphones.
Pocket Code sollte sofort bedienbar sein für jeden der Erfahrung mit Scratch hat. Zur Zeit (Juni 2016) gibt es Pocket Code noch nicht für Iphones sondern nur für Android.

Steve Holmes

Steve Holmes startete "Creative Computer Lab" 2009 um Kinder den Umgang mit Computern und computergesteuerten Maschinen zu lehren.

Steve konzentrierte sich auf Programmier-Anfänger und darauf Kindern die ersten Schritte in der Welt der computer zu ermöglichen. Er verwendet dazu Scratch und schrieb sein eigenes Scratch-Curriculum.

Steve's Schüler sind regelmäßig Finalisten bei der Irish National Scratch competition und haben u.a. 2011, 2012 und 2015 Preise gewonnen.

Für ältere Studenten entwickelte Steve einen Kurs "Einführung in die Webprogrammierung" mit einem Fokus auf gute Programmiertechniken, um weiterfürend erfolgreich zu sein in der Welt von phone/tablet apps und komplexeren Programmier-Umgebungen.

Da Steve gerne bastelt und immer Aussschau hält nach neuen Unterrichtsmethoden hat er folgende Erweiterungen für Scratch gebastelt:

  • Wii Balance Board and Wiimote
  • MakeSense Analog Sensor Board

Steve ist überzeugt dass Programmieren lernen auch folgende "life skills" fördert:

  • Problem solving
  • Creativity and innovation
  • Collaboration (Zusammenarbeit)
  • Grit and perserverence (Durchhaltevermögen)
  • Project ownership (Projektleitung)

Diese Fähigkeiten sind genauso wichtig wie technisches Fachwissen. Nicht alle Kinder werden später einmal als Programmierer arbeiten, aber sie werden obige Fähigkeiten sicher brauchen können.

Horst Jens

Horst JENS lebt in Wien, Österreich und bietet seit 2006 Programmierkurse für Kinder und Jugendliche an (http://spielend-programmieren.at). Er interessiert sich dafür Kindern und Jugendlichen die ersten Schritte in die Welt der Spiele-Programmierung und Publizierung zu ermöglichen. Ein großes Anliegen ist ihm, auf die Möglichkeiten der internationalen Zusammenarbeit und Publizierung in der Welt der freien Software aufmerksam zu machen, speziell im Bereich von free/libre open source games.
Er sieht im Bereich Spieleprogrammierung viel Potential um jungen Programmierer Werzeuge und Möglichkeiten in die Hand zu geben sich in der Welt der freien Software und internationaler Zusammenarbeit zu engagieren und sich dort kreativ und künstlerisch auszudrücken.
Sofern er nicht gerade damit beschäftigt ist auf seinen Computer Linux neu zu installieren interessiert er sich für freie Podcasts und Zeitschriftenprojekte
